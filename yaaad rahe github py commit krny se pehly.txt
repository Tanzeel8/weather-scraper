`.gitignore` ka system aise kaam karta hai:

* Agar aap **ek ek file manually commit** karte ho (`git add file_name`), to bhi

  * `.env`
  * `output/`

**track nahi hongi**, kyunki `.gitignore` me likhi hui hain.
